# 2.算法模式：
  ## 1.回溯      循环+递归        结束条件，循环，改变，递归，撤销                   穷举        自顶向下
  ## 2.动态规划   N重循环+循环     N重循环N维数组-状态转移方程（表格法）+循环选择比较   组合最优     自底向上
  ## 3.广度优先   双循环          外循环出队，内循环入队                           最少状态变化  由内向外
  ## 4.滑动窗口   双循环          外循环右指针向前，内循环左指针向前                 子串         循序渐进
